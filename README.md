# 🎵 AI Music Generation using LSTM

## 📌 Project Overview

This project uses Artificial Intelligence and Long Short-Term Memory (LSTM) neural networks to generate music from MIDI files. The model learns musical note sequences from a dataset and creates new melodies.

## 🚀 Features

* Read and preprocess MIDI files
* Extract musical note sequences
* Train an LSTM model
* Generate new AI-based music
* Save generated music as a MIDI file

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* PrettyMIDI
* NumPy
* Google Colab
* Git & GitHub

## 📂 Project Structure

```text
AI-Music-Generation/
│── dataset/
│── music_generation.py
│── train_model.py
│── generated_music.mid
│── requirements.txt
│── README.md
```

## ▶️ How to Run

1. Clone this repository.
2. Install the required libraries:

   ```
   pip install -r requirements.txt
   ```
3. Add MIDI files to the `dataset` folder.
4. Run `train_model.py` to train the model.
5. Run `music_generation.py` to generate new music.

## 📊 Output

* Extracts note sequences from MIDI files.
* Trains an LSTM model.
* Generates a new MIDI music file (`generated_music.mid`).
